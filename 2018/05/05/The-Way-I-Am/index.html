The Way I am

发表于 2018-05-05 分类于 编程人生

今天，终于上线了我的独立域名博客。

虽然还在上大学的时候，就开始在 CSDN 上写文章，但拥有自己的独立域名还是非常不一样的。从去年开始，萌生了想搭建个人站点的想法。然后从头开始学习前端，去年看完了 《Head First HTML and CSS》、《HTTP权威指南》，《JavaScript高级程序设计》也看了一大半。打算采用的技术栈是 Linux + Nginx + Python + Flask + BootStrap。但是去年年底有了换工作的想法，于是搭建博客的事情只能先放一放。

现在换工作的事情已经告一段落，于是搭建博客的事情重上日程。前几天了解到了一个静态博客引擎 Hexo，搭建个人博客比较简单，再配合 NexT 主题，博客界面的确很干净清爽。于是就先用了 Hexo，同时把博客内容部署在 Github 上，最后再申请一个域名，我的博客就展现在你们面前了。虽然博客上线了，但是我还是不想放弃之前的技术方案，因为那样才能真正锻炼我的前后端技术能力，所以 Hexo 方案只是暂时的。

趁着这个值得纪念的日子，也正好梳理一下我的北漂四年岁月。

关于浪潮

由于自己不是学计算机专业，而且当时又错过了校招的黄金时间，所以校招只拿到了浪潮的 offer。相对来说职业生涯的起点还是比较低的，但是也挺符合我当时的技术水平吧。现在回过头来看，当时做的一个决定挺正确的：工作地点选择了北京。

在浪潮只呆了一年，因为部门的技术水平的确很差。来公司没几天，我就想着尽快离开浪潮。于是每天晚上别的同事都下班了，我还呆在公司看书，一直看到晚上十点，周末也一个人来公司看书。所以那一年，个人技术水平有了质的飞跃，而且让我养成了一个保持到现在的习惯：一本一本看技术书。

在浪潮，也认识了几个很好的小伙伴：李延斌，挺不正经的一个人，但是关键时刻还是不会掉链子；芬芳姐，和你一起吐槽浪潮的日子真是快乐；还有续东，也不知道还在不在百度。

关于华三

从浪潮离职之后，面试的第一家公司就是华三，而且当天就给了 offer，当时面试官（也是我后来的老大）送我到电梯，说了一句让我至今印象深刻的话：“我就希望多一点像你这样的牛人来我们公司”。当时听的我挺激动的，毕竟得到面试官这样的认可，这也直接导致我没有动力去面试其他公司了（老大这个套路也不知道给多少个面试者用过～～）。

来到华三，我就直接维护了一个非常有技术含量的模块： LIPC。 LIPC 实现了华三 Cowmare 网络操作系统上的进程间通信功能，是一个纯内核模块。通过维护这个模块，我开始学习内核知识，在技术水平上又有了很大的提升。

在华三，有两件事情我挺自豪：

  • 维护 V7 LIPC 期间，解决了多个疑难 bug，得到了老大们的认可。在我来华三只有一年的时候，就得到了一个非常大的工作荣誉：华三研发金牌个人奖（后来在面试华为的时候，面试官觉得挺不可思议的，哈哈～～）；

  • 独立承担了 V9 LIPC 的内核开发，这包括读懂内核 TIPC 源代码，并在这个基础上进行比较大的扩展。这个项目的完成，让我不再仰视 V7 LIPC 的开发前辈们；

华三的 Comware 支撑组，牛人挺多的，离开华三还是挺不舍的。华三的两年多时光，充实而又快乐。老大一直挺照顾我的，给了我很多技术指点和人生建议。王鹏举奉行无为而治，作为我的直接 leader 以及 mentor，一直没怎么管过我，离职期间那天中午的散步聊天，真的让我感到很温暖。和勇哥合作做了一年的项目，勇哥的负责、耐心让我挺佩服的。和老刘、大方组成的中午散步三人相声组，也在老刘不甘心一直当捧哏后，宣告破产，以后机会就更少了。还有其它小伙伴们，也都已经走远。

祝愿华三能够发展顺利，毕竟真的有一群老员工在默默地支撑着、付出着。华三能够发展到现在，离不开这群人的坚持。

Juniper

今年 3 月，从华三办完了离职手续，开始了找工作之旅。一共面试了 3 家公司：华为、Juniper 和 Vmware。三家公司都给了 offer，最后基于自己的职业生涯考虑，选择了 Juniper。本来这次换工作，是冲着 BAT 去的，但是在拿到 Vmware offer 之后，就释然了。我觉得 Vmware 的 offer 已经能够证明自己过去三年多的积累了。

来 Juniper 的两周时间里，感觉还是非常不错的。最大的感受就是来到知识的海洋，可以任我畅游。身边的任何一个同事都是资深前辈，是我学习的对象。而且 manager 们也很 nice，dongdong 非常和善，而且很有追求，给我一种大哥的感觉。

关于未来

希望自己能够继续保持毕业以来的奋斗精神，继续钻研技术，保持学习的习惯。在内核、网络协议栈两个方向上继续深入研究，并学习网络虚拟化、容器等新技术。同时对前后端、分布式系统等领域也能有所了解。最后，身体是最重要的，跑步也要坚持下去。

最后，解释一下，文章标题为什么叫作 The way I am。之前看过许鞍华导演的一部电影 《天水围的日与夜》，这部电影是目前为止我最喜欢的一部电影，它的英文名字叫作 The Way We Are。
